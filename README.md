Este repositório reúne uma coleção completa de projetos desenvolvidos no Microsoft Excel, com foco em Análise de Dados, Business Intelligence (BI), automação de planilhas e resolução de problemas do mundo real.

Ao longo dos projetos são aplicadas desde funções básicas até recursos avançados do Excel, permitindo transformar dados brutos em informações estratégicas para tomada de decisão.

Conteúdos abordados
Limpeza e tratamento de dados
Organização e estruturação de bases de dados
Dashboards interativos
Relatórios gerenciais
Análises estatísticas
Indicadores (KPIs)
Tabelas e gráficos dinâmicos
Validação e formatação de dados
Automatização de cálculos

Objetivo

Demonstrar, na prática, como o Excel pode ser utilizado para análise de dados, criação de dashboards profissionais, automatização de processos e geração de insights para apoiar decisões em diferentes áreas de negócio.

Este repositório será atualizado constantemente com novos projetos, exercícios, estudos de caso e soluções utilizando diferentes recursos do Excel, servindo como material de estudo e também como portfólio para a área de Dados e Business Intelligence.
